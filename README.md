# SOCC Website

Minimal React + Vite scaffold for the SOCC website starter file.

## Structure

- `so_website.tsx`: original starter site file
- `src/main.tsx`: app entrypoint
- `src/App.tsx`: wrapper that mounts the starter file
- `src/styles.css`: global styles and responsive nav helpers
- `src/components/`: shared UI components for future splits
- `src/pages/`: page-level components for future splits
- `src/data/`: static content and mock data
- `src/assets/`: images, logos, and other bundled assets
- `public/`: static files served as-is
- `index.html`: Vite HTML entry
- `vite.config.ts`: Vite config

## Start

1. `npm install`
2. `npm run dev`
